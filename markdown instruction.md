## Списки

Чтобы добавить ненумированные списки, необходимо пункты выделить звездочкой (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумированные списки, необходимо пункты просто пронумировать. Например, вот так:
1. Первый пункт
2. Второй пункт
